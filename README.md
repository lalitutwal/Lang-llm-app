# Pet name generator using langchain
A lightning-fast name generator built with LangChain and Streamlit, powered by Groq's Llama-3 inference.

Setup

    Clone Repo: git clone https://github.com/lalitutwal/Lang-llm-app.git
    Install Deps: pip install streamlit langchain-groq langchain-core python-dotenv
    Environment: Add GROQ_API_KEY to your .env file.
    Run: streamlit run main.py
